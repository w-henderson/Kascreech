Kascreech: Privacy Policy

Effective as of August 13th, 2021.

William Henderson and Elliot Whybrow ("us", "we", "our") operate the Kascreech web application (the "Service").

By using the Service, you agree to the use of your data in accordance with this policy.

Introduction

We are committed to protecting the privacy of our users. This Privacy Policy outlines the ways in which we collect data about our users and how we process this data in accordance with data protection and security regulations.

The Service does not use the HTTPS protocol due to technical limitations, so some data is not encrypted during its transfer to our servers. However, this is no cause for concern since this data does not contain any personal or sensitive information and is protected as soon as it reaches its destination.

Request Data

"Request Data" refers to data transferred to our servers automatically by the browser upon visiting the Service. This data is transferred to every website you visit, and contains your IP address (or that of a VPN if one is in use), information about your browser and computer, and language preferences. Request Data is automatically removed from our servers as soon as your request or sequence of requests have been processed, with the exception of the IP address, which we log to prevent abuse of the Service. This data is only accessible by authorised individuals and is never shared with any third parties.

User Data

"User Data" refers to data shared with us directly by the user, and is comprised solely of the user name used to enter a game lobby. This data is only shared with the room host, as well as partially with participants in the room. User Data is removed automatically from our servers as soon as the game lobby closes, and would only persist in the event of a crash causing a memory dump. It is vital to the operation of the Service, and as a result, by using the Service, you agree to our use of User Data in accordance with this section of the Privacy Policy.

Advertising

The Service does not show any advertisements so no data is shared with advertisers. If the Service were to add advertisements in the future, this section would be updated to show how this would affect your data.
